# Telegram-бот «Азимут Клиник»

Бот для `@azimut_clinic_bot` повторяет спокойный тон сайта, помогает выбрать формат обращения, показывает услуги и цены, собирает минимальную заявку и передаёт её в amoCRM.

## Что умеет

- бережное меню без медицинских обещаний и самодиагностики;
- запись на консультацию: имя, телефон, направление, краткий комментарий;
- передача заявки в `Неразобранное → Обращения из чатов` amoCRM;
- AI-консультант с медицинскими ограничениями и экстренным маршрутом 112/103;
- резервная отправка заявки в администраторский Telegram-чат;
- запуск без внешних npm-зависимостей на Node.js 18+.

## Быстрый запуск

1. Скопировать `.env.example` в `.env` и задать переменные среды в хостинге.
2. Обязательно указать `TELEGRAM_BOT_TOKEN`.
3. Для amoCRM указать `AMO_LEAD_ENDPOINT`. Если endpoint защищён, также задать одинаковый `AMO_LEAD_API_KEY` в боте и Vercel.
4. По желанию добавить `DEEPSEEK_API_KEY` и `ADMIN_CHAT_ID`.
5. Проверить синтаксис и запустить:

```bash
npm run check
npm start
```

Токены не коммитить. Для постоянной работы polling-версии нужен Render, Railway, VPS или другой always-on процесс. Vercel используется для безопасного endpoint `/api/amo-lead`, где хранится токен amoCRM.

## Переменные amoCRM на Vercel

```text
AMOCRM_BASE_URL=https://lecmedea.amocrm.ru
AMOCRM_ACCESS_TOKEN=...
AMOCRM_PIPELINE_ID=11108006
AMOCRM_CHAT_STATUS_ID=87274334
AMOCRM_SITE_STATUS_ID=87274338
AMOCRM_FIELD_MAP={}
AMO_LEAD_API_KEY=...
```

Токен Telegram завтра нужно добавить только в секрет `TELEGRAM_BOT_TOKEN`; код менять не потребуется.
