# Единый реестр заявок Azimut Clinic

`Code.gs` — привязанный к рабочей Google-таблице серверный сборщик заявок.

Он:

- создаёт листы `Дашборд`, `Все заявки`, `Сайт`, `Telegram`, `amoCRM`, `Справочники`, `Справка`;
- принимает заявки только с корректным `LEADS_WEBHOOK_SECRET`;
- устраняет дубли по `requestId` и amoCRM lead ID;
- записывает UTM-метки и технические результаты доставки;
- отправляет уведомление о каждой новой заявке на `info@azimutclinic.ru`;
- строит KPI, таблицы источников и диаграммы на первом листе.

## Публикация

1. Открыть таблицу → **Расширения → Apps Script**.
2. Вставить `Code.gs`, сохранить и один раз выполнить `setupWorkbook`.
3. В свойствах скрипта создать `LEADS_WEBHOOK_SECRET` (не хранить его в GitHub).
4. Развернуть как Web App: выполнять от владельца, доступ — `Anyone`.
5. Добавить URL Web App и тот же секрет в Vercel как `LEADS_WEBHOOK_URL` и `LEADS_WEBHOOK_SECRET`.

Не публикуйте секрет, Telegram-токен или токен amoCRM в репозитории.
