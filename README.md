# azimut-medline-site

Статический сайт центра ментального здоровья "Азимут Клиник".

## Что внутри

- `index.html` - главная страница.
- `about.html`, `services.html`, `prices.html`, `doctors.html`, `blog.html`, `contacts.html`, `partnership.html`, `payment.html` - основные разделы.
- `article.html` - шаблон страницы статьи, наполняется через query-параметр `?slug=...`.
- `privacy.html`, `personal-data.html`, `terms.html` - юридические страницы.
- `css/` - общие стили, адаптив и стили аудиоплеера.
- `js/` - навигация, динамический рендер карточек, формы, блог, аудиоплеер и интеграционные адаптеры.
- `data/` - контентовые данные: услуги, цены, врачи, отзывы, статьи, аудиотреки.
- `assets/` - логотип, изображения и аудиофайлы.

## Как запустить локально

Проект не требует сборки.

```bash
python3 -m http.server 8000
```

После запуска открыть:

```text
http://localhost:8000/
```

## Интеграции заявок

- `api/amo-lead.js` принимает заявки сайта и Telegram, создаёт контакт/сделку в amoCRM и независимо зеркалирует обращение в Google Sheets с email-уведомлением.
- `api/amo-webhook.js` принимает защищённые события amoCRM для заявок, созданных вручную в CRM.
- `google-apps-script/Code.gs` содержит сервер реестра и настройку дашборда.
- `telegram-bot/` содержит код `@azimut_clinic_bot`; токен задаётся только в окружении сервера.
- `assets/vk-2026/` — актуальный пакет из 13 изображений VK с корректными размерами и безопасными зонами.

Шаблон обязательных серверных переменных находится в `.env.example`. Секреты нельзя коммитить в GitHub.
- Подключить реальную карту на странице контактов вместо `.map-placeholder`.
- Подключить Bitrix24 вместо `js/bitrix24-placeholder.js`.
- Подключить платёжного провайдера вместо `js/payment-placeholder.js`.
- Финальные медицинские формулировки требуют юридического и врачебного согласования перед публичным релизом.

## Быстрые проверки

Проверить статус страниц:

```bash
for page in index.html services.html doctors.html prices.html blog.html article.html contacts.html payment.html privacy.html terms.html personal-data.html partnership.html about.html; do
  code=$(curl -s -o /dev/null -w '%{http_code}' "http://localhost:8000/$page")
  printf '%-20s %s\n' "$page" "$code"
done
```
