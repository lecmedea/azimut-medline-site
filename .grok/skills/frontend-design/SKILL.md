# Frontend Design (Azimut Clinic)

Brand context: mental health clinic «Азимут Клиник» — calm, premium, warm onyx/cream palette, compass motif, medical trust without cold corporate feel.

## Token reference

- H1: #584021, Montserrat Regular
- Accent gold: #AE8B66, headings #977959
- Backgrounds: onyx-warm-bg, cream gradients
- Buttons: #AE8B66

## Process

1. Ground changes in clinic context — care, orientation, 24/7 availability.
2. One signature element per section; keep surrounding layout disciplined.
3. Align grid columns, footer zones, and type scale before adding decoration.
4. Critique spacing with screenshots when possible.
