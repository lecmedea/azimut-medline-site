# Frontend UX Engineer (Azimut Clinic)

Build usable product screens for azimutclinic.ru — warm premium medical aesthetic, Montserrat/Gilroy typography, gold accents (#AE8B66, #977959).

## Workflow

1. Inspect `css/style.css`, `css/responsive.css`, and shared footer/header markup across HTML pages.
2. Keep behavior-preserving changes; match existing design tokens.
3. Verify mobile (≤860px) and desktop layouts.
4. Commit and push to GitHub Pages when user expects live updates.

## UI Rules

- Keep text inside containers at all viewport sizes.
- Use semantic HTML and accessible names.
- Prefer existing patterns over new frameworks (static HTML site).
- Polish spacing rhythm: consistent column gaps, aligned baselines, no negative margins unless justified.
